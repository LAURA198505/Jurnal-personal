
Frontend – Jurnal Personal

Descriere
Frontend-ul aplicației Jurnal Personal reprezintă interfața prin care utilizatorul poate crea, vizualiza și gestiona însemnări într-un jurnal digital. Aplicația comunică cu backend-ul prin cereri HTTP către un API REST.

Funcționalități
- formular pentru adăugarea unei însemnări
- afișarea listei de însemnări existente
- ștergerea unei însemnări
- comunicare asincronă cu backend-ul

Tehnologii utilizate
React.js
JavaScript
HTML
CSS

Pornirea aplicației
1. Instalează dependențele: npm install
2. Pornește aplicația: npm start
Aplicația va rula implicit la adresa http://localhost:3000

Observație
Frontend-ul este realizat în scop educațional și respectă principiile de bază ale unei aplicații web de tip client–server.
